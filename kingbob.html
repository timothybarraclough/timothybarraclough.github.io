king bob

2014 -

King Bob is a music / art project collaboration between Jonathan Shirley and Timothy J. Barraclough.
The project website is located at http://king-bob.com and is host to various EPs, videos and the Motion Sickness Audio Unit.

Motion Sickness Audio Unit

Motion Sickness is an audio unit designed to emulate vinyl warble using a single delay line. It contains various glitches and artifacts that are programmatic bugs. These were left in the finished audio unit in order to preserve the state in which it was used for the King Bob project. It was programmed with Juce in C++. Source can be made available upon request.
